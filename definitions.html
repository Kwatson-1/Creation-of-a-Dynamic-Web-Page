Programming concept definitions

Authentication: is a means by which a server will attempt to identify who is accessing their information or site. It is also used by users to determine whether a server is the system it claims to be.

Web security: refers to the protection of users, computer systems and networks from damage to or theft of software, hardware, or data.

HTTP: is the set of rules for transferring files like text, images, sound, video, and other multimedia files over the web. HTTP follows a client-server model where a client opens a connection to make a request and will then wait until it receives a response.

Session management: is the process by which multiple requests to a web-based application or service from a single user or entity are securely handled. Websites use HTTP to communicate and a session is series of HTTP requests and transactions initiated by the same user.

Stateless programming: refers to a program or application that ‘does not maintain a state‘ or when the infrastructure of a system prevents a program from maintaining state. This means that data is not stored or carried from one session to the next.

1. Sequence logic, or sequential flow: refers to the sequential ordering of execution for blocks of instructions.

2. Selection logic, or conditional flow: is also referred to as alternation or conditional branch in which the algorithm must select one of alternating paths to follow.

3. Iteration logic, or repetitive flow: is also referred to as loops, which is a set (or block) of instructions that are executed zero, one, or more times.

HTML: is a formatting system for displaying material retrieved over the internet. Each retrieval unit is known as a Web page which often contains hypertext links that allows related pages to be retrieved.

Markup language: a markup language is a computer language that uses tags to define elements within a document. Being human readable, markup files contain standard words instead of typical programming syntax. An example of some basic tags include <head>, <body> and <div>.

CSS: is used for describing the presentation of a document written in a markup language like HTML or XML. It can be used to define styles for multiple websites at once including their design, layout and variation in display to facilitate different devices and screen sizes.

Syntax: places emphasis on investigating a problem and its requirements for the purpose of gathering information to inform a conclusion or recommend a specific course of action.

Programming languages: are tools used to bridge human thoughts into instructions for computers. Computers understand the base-2 numeral system (0’s and 1’s) as a language which by using a programming language can be translated into a form readable for humans to more easily understand and write.
